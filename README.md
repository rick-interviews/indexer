## Instructions

### Microservices

1. API

Start by using the following command:

```
node ./api/indexer_api.js 8083
```

2. Analyzer

Start by using the following command:

```
node ./analyzers/ethereum/eth_analyzer.js 8082
```

3. Scanner

Start by using the following command:

```
node ./scanners/ethereum/eth_scanner.js 8081
```
